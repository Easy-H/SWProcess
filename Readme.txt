# 실행 방법
1. Unity Hub 설치
2. Unity Editor 2021.3.24f1 설치
3. Unity Hub에서 프로젝트 Open
	3.1. Unity Editor를 2021.3.24f1와 다른 버전으로 설치하면, Hub에서 Editor viersion을 바꿔 실행
4. 프로젝트 창(Default Layout 기준 아래 창)의 여러 요소 중 PlayScene을 더블클릭하여 열기
5. Unity Editor 위의 실행 버튼을 눌러 플레이

## Play
1. 처음 실행한 화면에서 Play 버튼을 눌러 레벨 선택 GUI으로 이동
2. 레벨 선택 GUI에서 수식을 터치하여 Play GUI으로 이동
3. Play GUI에서 유닛을 누른 상태로 마우스를 드래그하여 유닛의 위치를 옮김
	3.1. 가운데의 정지 버튼을 터치하여 잠시 게임을 정지할 수 있음
		3.1.1. 왼쪽의 재시작 버튼을 누르면 레벨 초기화
		3.1.2. 가운데의 계속하기 버튼을 누르면 Pause 종료
		3.1.3. 오른쪽의 버튼을 누르면 레벨 선택 GUI로 이동
	3.2. 왼쪽 화살표 버튼을 터치하여 이동을 취소
	3.3. 오른쪽 화살표 버튼을 터치하여 취소한 이동을 취소
4. 유닛을 옮겨 화면의 수식이 모두 올바르다면 클리어 GUI가 나타남
	4.1. 가운데의 계속하기 버튼을 눌러 다음 레벨을 플레이 할 수 있음
	4.2. 왼쪽의 재시작 버튼을 눌러 레벨을 한번 더 플레이 할 수 있음

## Build
1. 처음 실행 화면에서 Build 버튼을 눌러 레벨 제작 화면으로 이동
2. 레벨을 제작
	2.1. 유닛 추가하기
		2.1.1. 왼쪽 아래의 연필 버튼을 눌러 추가할 유닛 선택창 Open
		2.1.2. 추가할 유닛을 선택한 후 추가하고자 하는 위치를 클릭하면 클릭한 위치로 새로운 유닛 추가
	2.2. 유닛 삭제하기
		2.2.1. 연필 버튼 옆의 쓰레기통 버튼을 누름
		2.2.2. 삭제하고자 하는 유닛을 클릭하면  그 유닛 삭제
	2.3. 가운데의 정지 버튼을 눌러 Build를 잠시 정지할 수 있음
3. 오른쪽 아래의 공유 버튼을 누른 후 현재 레벨을 플레이
	3.1. 정지 버튼을 누르고 연필 버튼을 누르면 플레이를 중단하고 레벨 에디트 GUI로 돌아감
4. 유닛을 옮겨 화면의 수식이 모두 올바르드면 클리어 GUI가 나타남
	4.1. 업로드 버튼을 누르면 Firebase에 현재 레벨 정보를 올림(Todo)
