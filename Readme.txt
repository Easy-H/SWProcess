# 실행 방법
1. Unity Hub 설치
2. Unity Editor 2021.3.24f1 설치
3. Unity Hub에서 프로젝트 Open
	3.1. Unity Editor를 2021.3.24f1와 다른 버전으로 설치하면, Hub에서 Editor viersion을 바꿔 실행
4. Unity Editor 위의 실행 버튼을 눌러 플레이

# 미구현사항
1. Firebase를 이용한 커스텀 레벨 업로드
2. 커스텀 레벨 플레이

## Play
1. 처음 실행한 화면에서 Play 버튼을 눌러 레벨 선택 화면으로 이동
2. 레벨 선택 화면에서 수식을 터치하여 Play 화면으로 이동
3. Play 화면에서 유닛을 누른 상태로 마우스를 드래그하여 유닛의 위치를 옮김
	3.1. 가운데의 정지 버튼을 터치하여 잠시 게임을 정지할 수 있음
	3.2. 왼쪽 화살표 버튼을 터치하여 이동을 취소
	3.3. 오른쪽 화살표 버튼을 터치하여 취소한 이동을 취소
4. 유닛을 옮겨 화면의 수식이 모두 올바르다면 클리어 화면으로 넘어감
	4.1. 다음 스테이지로 이동할 수 있음
	4.2. 현재 스테이지를 다시 플레이할 수 있음
